Problem Statement

📍 Background

Traditional Men's Wear (Kurtas, Sherwanis, Nehru Jackets) is a ₹40,000+ Crore industry in India. Small boutiques like Scandal Men's Wear, Thane have loyal customers but no digital presence.

  • No online ordering — customers must visit the store physically
  • No size guidance — wrong purchases cause returns
  • No visibility beyond local area; growth is limited
  • No order tracking or flexible payment options
  • No personalized style recommendations for customers

💡 Need for the System

A full-stack, responsive e-commerce web platform to digitize the complete shopping experience — from product browsing to secure checkout, order tracking, and personalized recommendations.

🌿

SDG 8 – Decent Work & Economic Growth

Empowering a small Indian traditional wear business to scale digitally, reach a wider audience, and grow sustainably through technology.

Project Objectives
1
Design a responsive, premium web interface using HTML5, CSS3 with custom CSS variables, and Vanilla JavaScript — featuring dark/light mode, animations, and elegant typography.
2
Implement secure user authentication (Register & Login) using JWT tokens, bcrypt password hashing, and session-based access control via Node.js & Express.js.
3
Integrate a PostgreSQL database to manage Users, Orders, and Reviews with full CRUD operations and proper relational schema design.
4
Implement a complete checkout system with Razorpay payment gateway integration (online) and Cash on Delivery (COD) as an alternative.
5
Build smart user-assist features: an AI Wardrobe Stylist (occasion-based outfit curation) and an Interactive Size Consultant (height/weight-based size recommendation with confidence rating).

Tools & Technologies Used

🎨 Frontend

  • HTML5 – Semantic structure & SEO
  • CSS3 – Custom variables, animations, glassmorphism
  • Vanilla JavaScript – DOM, LocalStorage, fetch API
  • Google Fonts – Playfair Display, DM Sans

🔧 Backend

  • Node.js + Express.js – REST API server
  • PostgreSQL – Relational Database (SQL)
  • pg / Sequelize – Database connectivity (ORM/Driver)
  • JWT – JSON Web Token authentication
  • bcryptjs – Password hashing & security

💳 Integrations

  • Razorpay – Live payment gateway (UPI/Card/Net Banking)
  • Cash on Delivery – COD payment option

System Architecture

Browser (HTML/CSS/JS)
    ⬇ fetch() REST API calls
Express.js Server (Node.js)
    ⬇ pg / Sequelize (SQL queries)
PostgreSQL Database
    + JWT Auth Layer
    + Razorpay SDK
